What is mean total number of steps taken per day?

I will determine the number of steps taken each day by looping through each day, collect all the “steps” values for each 5 min interval in a list. The resulting list will contain 61 vectors (one for each day in the data set) where each vector contains 288 “step” values for each 5 min interval in the 24 hour day. I will collect sum total number of steps that were taken for each day in a “totals” vector.

steps_day <- list()
totals <- vector(mode = "numeric")
dates <- levels(data$date)
for (i in 1:length(dates)) {
    steps_day[[i]] <- data[data$date == dates[i], "steps"]
    totals[i] <- sum(steps_day[[i]], na.rm = TRUE)
}

First I will plot a histogram of the total number of steps taken each day.
